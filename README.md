# Renato Craveiro - Portfolio

My personal portfolio website showcasing my work as a Software Developer and Infrastructure Technician.

## Overview

Welcome to my portfolio! This site is a curated collection of my professional work, experience, and insights. I share my highlighted projects, career journey, and thoughts on technology and development.

## What's Inside

- **Projects**: Showcase of my full-stack projects including BulkMailer, SmartCompassTracker, and NoteTakingApp
- **Experience**: Professional work history and roles
- **Blog**: Technical articles and insights on development, AI, and infrastructure
- **Connect**: Social media links and contact information

## Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Motion for smooth, interactive components
- **Content**: MDX for blog posts



## Deployment

The portfolio is deployed and live. You can visit it to explore my work and connect with me.
